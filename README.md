# PrismIDE
This Project is coded by Python Tkinter.

## *What It Can Do*
- Edit
  + New
  + Open
  + Save
  + Save as...
- File
  + Undo
  + Redo
  + Copy
  + Cut
  + Paste
  + Select all

It also have the **RightClickMenu**.

And there're a line number bar that help you to write clearly.

## *How It Works*
All things are ran in *```main.py```*.
